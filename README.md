## ✨ Features

- 🎨 **Design Premium** — Dark theme com glassmorphism, gradientes e micro-animações
- ⌨️ **Typewriter Effect** — Animação de digitação no Hero alternando entre roles
- 📱 **100% Responsivo** — Adaptado para mobile, tablet e desktop
- 🔍 **SEO Otimizado** — Meta tags, heading hierarchy e semantic HTML
- ⚡ **Performance** — Scroll-triggered animations com IntersectionObserver
- 🧩 **Arquitetura Modular** — Componentes organizados por responsabilidade

## 🏗️ Estrutura do Projeto

```
src/
├── components/
│   ├── ui/                     # Componentes reutilizáveis
│   │   ├── AnimatedSection.jsx     ← Animação de scroll
│   │   ├── GlassCard.jsx           ← Card com glassmorphism
│   │   ├── SectionTitle.jsx        ← Título de seção
│   │   └── SkillBadge.jsx          ← Badge de habilidade
│   └── sections/               # Seções do portfólio
│       ├── Navbar.jsx              ← Navegação com scroll spy
│       ├── Hero.jsx                ← Seção principal
│       ├── About.jsx               ← Sobre mim + Formação
│       ├── Skills.jsx              ← Habilidades técnicas
│       ├── Projects.jsx            ← Projetos
│       ├── Experience.jsx          ← Timeline de experiência
│       ├── Certifications.jsx      ← Certificações (accordion)
│       ├── Contact.jsx             ← Contato
│       └── Footer.jsx              ← Rodapé
├── data/
│   └── personalData.js        # Dados centralizados do currículo
├── App.jsx                    # Layout principal
├── App.css                    # Animações auxiliares
├── index.css                  # Design system + utilities
└── main.jsx                   # Entry point
```

## 🚀 Como Rodar

```bash
# 1. Clone o repositório
git clone https://github.com/Nicolas125-tech/Portifolio-em-React-e-Talwind.git

# 2. Entre na pasta
cd Portifolio-em-React-e-Talwind

# 3. Instale as dependências
npm install

# 4. Rode o servidor de desenvolvimento
npm run dev
```

> O app estará disponível em `http://localhost:5173`
